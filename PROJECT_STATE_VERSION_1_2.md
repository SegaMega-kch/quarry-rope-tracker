## Канаты

В интерфейсе используются короткие названия:

- `41м (ø45)`
- `110м (ø45)`
- `40м (ø52)`
- `82м (ø52)`

Полные типы в базе:

- `Напор ЭКГ-10`
- `Подъём ЭКГ-10`
- `Напор ЭКГ-12К`
- `Подъём ЭКГ-12К`

Правила:

- `82м (ø52)` однозначно относится к подъему ЭКГ-12К.
- Канаты учитываются штуками, не остатками метров.
- На одной вертушке максимум 2 каната.

## Вертушки

Вертушки являются отдельной сущностью.

Всего создано 6 вертушек:

- `Вертушка №1`
- `Вертушка №2`
- `Вертушка №3`
- `Вертушка №4`
- `Вертушка №5`
- `Вертушка №6`

Логика:

- вертушка может быть пустой;
- вертушка может быть загруженной;
- у вертушки есть текущее место;
- канат на вертушке всегда должен быть привязан к конкретной вертушке;
- занятую вертушку нельзя случайно использовать в другом месте;
- пустую вертушку можно перемещать отдельно;
- загруженную вертушку можно переместить на доступное место;
- список мест для перемещения вертушки показывается вертикально: сначала экскаваторы, потом ПП, потом остальные места.

Карточка вертушки сейчас акцентирует:

1. какой канат;
2. где находится вертушка;
3. номер вертушки и загрузку.

Пример:

```text
41м (ø45) - 1
ЭКГ-8И №42
Вертушка №1 • 1/2
```

## Под 30т краном

В блоке `Под 30т краном` есть две основные кнопки:

- `На вешелах`
- `На земле`

Кнопка `На вертушках` была убрана, потому что вертушки теперь вынесены в отдельный блок.

Во всплывающем меню добавления на вешела/землю:

- строки компактные;
- поле количества сдвинуто левее;
- кнопки `+` и `-` расположены горизонтально;
- сначала `+`, потом `-`;
- фон меню слегка отличается от белого.

## Добавление каната на вертушку

Добавление каната на вертушку делается из карточки конкретной вертушки кнопкой `Добавить канат`.

Меню оформлено карточками:

- сверху тип каната;
- ниже откуда он берется, например `30т кран, На вешелах`;
- справа поле количества;
- рядом кнопка `Переместить`;
- цвет кнопки `Переместить` сделан менее насыщенным.

Логика:

- в списке показываются доступные канаты;
- нельзя выбрать текущую же вертушку как источник;
- нельзя загрузить больше 2 канатов на одну вертушку;
- если вертушка уже занята в другом месте, она не должна принимать канат из другого места.

## Установка каната

Установка теперь делается через карточку вертушки.

Если вертушка стоит у экскаватора и на ней есть канат:

- появляется кнопка `Установить`;
- если на вертушке несколько канатов, открывается меню выбора;
- по установке списывается 1 канат с вертушки;
- такой же канат появляется как б/у на земле у этого экскаватора;
- история установки сохраняется.

Старая кнопка `УСТ` на обычных карточках остатков убрана.

## Б/у вывезти

Создан отдельный блок `Б/у вывезти`.

Он появляется только если есть канаты со статусом:

```text
USED_NEAR_EXCAVATOR
```

В блоке:

- карточки б/у канатов выделены контуром;
- есть кнопка `Вывезти`;
- по нажатию канат уходит из текущего подсчета в статус `WRITTEN_OFF`;
- история вывоза сохраняется.

## Откат

Откат работает по действию мастера, а не по отдельной строке истории.

Для этого в `RopeMovement` добавлено поле:

```text
operationId
```

Правила:

- одно действие пользователя может создать несколько строк истории;
- все строки одного действия имеют общий `operationId`;
- откат откатывает действие целиком;
- доступен откат последних 3 действий текущего пользователя;
- верхняя кнопка `Откатить` откатывает последнее доступное действие текущего пользователя;
- в истории кнопка `Откатить` показывается только один раз на одно действие.

Отдельно поддержан откат:

- добавления;
- корректировки остатка;
- перемещения;
- перемещения вертушки;
- установки;
- добавления б/у;
- списания/вывоза.
