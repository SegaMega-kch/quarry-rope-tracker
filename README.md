# Учёт канатов на карьере

Мобильное веб-приложение на Next.js, React, TypeScript, Prisma и SQLite для учёта канатов экскаваторов.

## Запуск

```powershell
npm install
npm run db:push
npm run db:seed
npm run dev
```

Откройте в браузере телефона адрес компьютера в локальной сети, например `http://192.168.1.10:3000`.

## Что есть

- остатки по типу, диаметру, длине, месту, размещению и статусу;
- добавление, перемещение, установка, учёт б/у каната, списание;
- история всех действий;
- справочник мест для кладовщика;
- заявки механикам для начальника;
- Excel-отчёты: текущий остаток и движение за месяц.
