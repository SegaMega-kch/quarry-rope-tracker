# ЯКНО: локальная реализация

Дата: 14.07.2026

## Что добавлено

- Новая ветка приложения `/yakno`.
- Новая вкладка в верхнем меню: `ЯКНО`.
- Экскаваторы берутся из общего справочника мест с категорией `excavator`.
- Горизонты берутся из существующего справочника горизонтов ветки `Сборки`.
- Добавлены отдельные блоки:
  - карточки экскаваторов;
  - `Свободные ЯКНО`;
  - `ЯКНО в ремонте`;
  - `Справочник ЯКНО` для кладовщика;
  - `История ЯКНО`.

## Новые таблицы Prisma

- `YaknoBox` - сами ящики ЯКНО.
- `YaknoExcavatorState` - выбранный горизонт для экскаватора во вкладке ЯКНО.
- `YaknoMovement` - история действий и данные для отката.

## Логика

- У экскаватора может быть только один запитанный ЯКНО.
- Запитанный ЯКНО отображается первым и получает подпись `зап`.
- Остальные ЯКНО возле экскаватора отображаются строками ниже.
- Свободному ЯКНО можно указать горизонт.
- ЯКНО в ремонте нельзя выбрать для экскаватора.
- Кладовщик может добавлять ЯКНО, удалять свободные ЯКНО из справочника, отправлять в ремонт и возвращать из ремонта.
- Добавлен откат последних 3 действий текущего пользователя для ветки ЯКНО.
